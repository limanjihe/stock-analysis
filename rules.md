# A股交易机会筛选规则

## 市场

- 只分析 A 股（沪深主板、创业板、科创板、北交所），不分析美股或其他海外市场

## 数据源

- 东方财富 API（主力）：`82.push2.eastmoney.com`（实时行情）、`push2his.eastmoney.com`（K线历史）
- akshare：备用，需注意代理冲突（直连东方财富更稳定）
- yfinance：仅美股，A股场景不用
- 代理 `127.0.0.1:18095`：访问海外数据源时用；东方财富 API 直连即可，走代理反而容易断
- K线接口 `push2his` 不稳定时，用 `82.push2` 批量实时行情接口替代，通过今日OHLC推算技术指标

## 选股标准

- 排除涨停板（买不进），聚焦涨幅 3-10% 区间
- 必须有流动性：成交额 >2 亿，换手率 >2%
- 排除 ST、退市风险股
- 核心寻找「市场定价错误」而非单纯看涨：估值错配、叙事滞后、预期差

## 输出格式

每只票必须包含：
1. **进场区间**（具体价位+理由）
2. **做错了怎么办**（止损价位+亏损幅度）
3. **止盈区间**（目标价位+盈利幅度）
4. **R:R 比率**（风险收益比，至少 1:2）
5. **定价错在哪**（一句话说清市场在哪里定价错了）

## 核心纪律

- 不追高：今天涨的票必须等回踩才能进场
- 优先选量比>1.5 且回踩不放量的标的
- 不提「显而易见、人人都已进场」的机会
- 5 只票，不是凑数，是真的有边际优势
